# KORGAN

Korgan, Türkçe'de "KALE" anlamına gelir. Bu repoya bu adı vermemin sebebi problem çözerken sığınacağımız bir kale niteliği taşıdığı içindir. Bu çalışmamda [CTF(Capture The Flag)](https://en.wikipedia.org/wiki/Capture_the_flag#Computer_security) için derlemiş olduğum araçlar, yardımcı uygulamalar, kütüphaneler kısaca CTF için her şeyi barındıran bir liste şeklinde repodur.
